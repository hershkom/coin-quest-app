# כספת המטבעות — Android APK

## הוראות בנייה (5-10 דקות)

### שלב 1: התקן Android Studio
1. הורד מ: https://developer.android.com/studio
2. התקן (ברירות מחדל — אין צורך לשנות כלום)

### שלב 2: פתח את הפרויקט
1. פתח Android Studio
2. לחץ **"Open"**
3. נווט לתיקיית **`android-app`** (לא ל-`android-app/app`, אלא לתיקיה הראשית)
4. לחץ **OK**
5. חכה ש-Android Studio יטען את הפרויקט (30-60 שניות)

### שלב 3: בנה APK
1. בתפריט העליון: **Build → Build Bundle(s) / APK(s) → Build APK(s)**
2. חכה ~2 דקות
3. כשמסיים — תראה הודעה "Build successful"
4. לחץ **"locate"** כדי לפתוח את התיקייה עם ה-APK

### שלב 4: העבר לטלפון
**אפשרות א (כבל USB):**
1. חבר את הטלפון למחשב
2. העתק את ה-APK לטלפון
3. פתח ב-File Manager והתקן

**אפשרות ב (WhatsApp/Email):**
1. שלח את ה-APK לעצמך
2. פתח בטלפון והתקן

### שלב 5: הפעל התקנה מ-מקורות לא ידועים
1. בטלפון: **הגדרות → אפליקציות → הגדרות מיוחדות → התקן אפליקציות לא ידועות**
2. אפשר לאפליקציה שממנה אתה מתקין
