## 1. חשבון מפתח Google Play
1. עבור ל-[play.google.com/console/signup](https://play.google.com/console/signup), שלם 25$ חד-פעמי.
2. מלא פרטי מפתח (שם שיוצג לציבור, פרטי קשר).

## 2. בניית ה-AAB לחתימה
```bash
cd android-app
./gradlew bundlePlayRelease
# הפלט: app/build/outputs/bundle/playRelease/app-play-release.aab
```
**חשוב (לקח מפרויקט אחר, tap_runner):** `minifyEnabled` כבוי כרגע (`build.gradle`) — אם תפעיל
R8 בעתיד, התקן sideload ובדוק שהאפליקציה עולה לפני שמעלים ל-Play. שם עצירה מוחלטת בעבר.

## 3. יצירת האפליקציה בקונסולה
1. **Create app** → שם: "כספת המטבעות" / "Coin Quest" → שפת ברירת מחדל: עברית.
2. **Category:** Parenting / Education (בהתאם למה שיוצע).

## 4. Data Safety (טופס בטיחות נתונים) — תשובות מוכנות
בהתבסס בדיוק על מה שנאסף בפועל (ראו privacy.html):
| שאלה | תשובה |
|---|---|
| האם האפליקציה אוספת/משתפת נתוני משתמש? | כן |
| סוגי נתונים | Email (חשבון Google של ההורה), User IDs (Firebase UID), App activity (נתוני משפחה: מטלות/מטבעות/היסטוריה) |
| האם הנתונים מוצפנים בהעברה? | כן (Firebase HTTPS) |
| האם ניתן לבקש מחיקת נתונים? | כן — קישור: `https://coin-quest-app.firebaseapp.com/delete-account.html` |
| האם הנתונים משותפים עם צד ג'? | לא, למעט אם ההורה מפעיל ידנית תכונת צ'אט AI אופציונלית (Groq) — לסמן כ"אופציונלי/בשליטת המשתמש" |
| מטרת האיסוף | App functionality בלבד (לא פרסום, לא אנליטיקס) |

## 5. הצהרת קהל יעד ומדיניות Families
1. **Target audience:** סמן שהאפליקציה מתאימה גם לילדים (ראו S4 ב-ANDROID-APP-PLAN.md).
2. **דירוג תוכן:** למלא את שאלון ה-IARC — אין אלימות/תוכן בוגר, ציפוי צפוי: "לכל הגילאים".
3. לוודא שתיאור החנות **לא** כולל טענות טיפוליות/רפואיות ("מותאם ל-ADHD/אוטיזם" זה בסדר
   כתיאור התאמה, "מטפל ב-" / "עוזר רפואית ל-" זה לא בסדר — ראו S8 המקורי ב-ANDROID-APP-PLAN.md).
4. **ללא פרסומות** בגרסה זו — מפשט את שאלון ה-Ads.
